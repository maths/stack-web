# STACK Professionals' Network: Meeting on 29 May 2025

The meeting was hybrid as part of the Trieste STACK conference. 

## Developers Updates
Edmund gave an update of the Moodle-STACK release. There was a release in April, which included the XML editing of questions and the new parser. There will be another release in 3-4 weeks. This will include the new STACK dashboard, the metadata entry (or there will be another release shorlty after regarding the metadata) and potentially the free text entry questions. 

Raul gave an update of the ILIAS-STACK. There will be an update to the new core maybe during summer or September. 


## Items Discussed

### JSXGraph examples: 
Konstantina collected questions from HELM and FAC, that have JSXGraphs for the topics of Algebra and Calculus. Will import them in the STACK library. There is a Zulip thread [here](https://stack-assessment.zulipchat.com/#narrow/channel/384527-visualization/topic/JSX.20Graph.20examples.20in.20STACK.20Library/with/598562923), where people can suggest what examples of JSXGraph thay want to include in the library.

### New members:
New STACK users are welcome to join the STACK Professionals Network, there is no need to be experts. New users can ask for support in ZulipChat [https://stack-assessment.zulipchat.com/](https://stack-assessment.zulipchat.com/).
